About STREAM
=============================
This is a React Native application built with Expo and NativeWind/Tailwind CSS. The app allows users to browse and search for streaming movies. It features a tab-based navigation system, a search bar, and displays a list of the latest movies as well as trending movies.
Features
--------
- Tab-based navigation using React Navigation.
- Search functionality to find movies.
- Display of latest movies and trending movies.
- Responsive design with NativeWind/Tailwind CSS.
Getting Started
---------------
To get started with the app, follow these steps:
1. Clone the repository:
   ```bash
   git clone   
   ```
2. Navigate to the project directory:
   ```bash
   cd react-native-expo-nativewind-tailwind
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the Expo development server:
   ```bash
   npm start
   ```
5. Use the Expo Go app on your mobile device or an emulator to run the app.
Dependencies
------------
- React Native
- Expo
- NativeWind
- Tailwind CSS
- React Navigation
- Axios (for API requests)
